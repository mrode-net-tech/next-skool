# Week 9 — React Basics

**Goal:** Bootstrap `my-web` with Vite + TypeScript + React 18. Learn the core primitives: components, props, state, effects, and your first RTL test.

## Outcome

A `my-web/` project with:
- Vite dev server running at `http://localhost:5173`
- A `TaskList` component rendering a hardcoded array of tasks
- A `NewTaskForm` component with controlled inputs and a submit handler
- `useEffect` fetching tasks from `my-api` and storing them in local state
- First RTL test asserting that tasks render from a mocked response
