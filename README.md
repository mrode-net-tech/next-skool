# next-skool

A personal **24-week self-learning plan** for transitioning from **Laravel/PHP** to **modern fullstack TypeScript** (Node.js + React + Next.js).

The plan culminates in building a real production-ready application: **Habit Tracker SaaS**.

## Quick start

1. Open [`learning-plan/README.md`](./learning-plan/README.md)
2. Begin with **Day 1** in [`learning-plan/phase-1-foundations/week-01-setup/day-01-install-tools.md`](./learning-plan/phase-1-foundations/week-01-setup/day-01-install-tools.md)
3. Track your progress in [`PROGRESS.md`](./PROGRESS.md)
4. Write your code in [`exercises/`](./exercises/)

## Capstone project

The plan culminates in building **Habit Tracker SaaS** — a real production-ready application with auth, jobs, monitoring, CI/CD and Docker. See [`learning-plan/capstone-project/`](./learning-plan/capstone-project/).

## Stack

TypeScript, Node.js, Express, Prisma, PostgreSQL, Zod, JWT, React, Vite, React Router, TanStack Query, react-hook-form, Tailwind CSS, shadcn/ui, Next.js, Auth.js, Vitest, Supertest, React Testing Library, Playwright, MSW, Docker, GitHub Actions, BullMQ, Redis, Sentry, pnpm workspaces, Turborepo, Railway, Vercel.
