# next-skool

A 24-week self-learning plan for transitioning from Laravel/PHP to modern fullstack TypeScript (Node.js + React + Next.js).

> 🚧 The full learning plan is being generated. Once ready, see [`learning-plan/README.md`](./learning-plan/README.md) to start.

## Quick start

1. Open [`learning-plan/README.md`](./learning-plan/README.md)
2. Begin with **Day 1** in `learning-plan/phase-1-foundations/week-01-setup/day-01-install-tools.md`
3. Track your progress in [`PROGRESS.md`](./PROGRESS.md)
4. Write your code in [`exercises/`](./exercises/)

## Capstone project

The plan culminates in building **Habit Tracker SaaS** — a real production-ready application using Next.js, Prisma, BullMQ, Docker, and more. See `learning-plan/capstone-project/` once generated.
